## üéØ Common Tasks

### Setup
1. Read [START-HERE.md](./START-HERE.md)
2. Run `.\verify-setup.ps1`
3. Run `.\setup-complete.ps1`

### Development
1. Run `npm run dev`
2. Visit http://localhost:3100
3. Edit files and see changes

### Mobile
1. Run `npm run build:mobile`
2. Run `npx cap sync android`
3. Run `npx cap open android`

### Deployment
1. Deploy Lambda: `cd aws-lambda && .\deploy-to-aws.ps1`
2. Update `.env.local` with API URL
3. Build mobile: `npm run build:mobile`
