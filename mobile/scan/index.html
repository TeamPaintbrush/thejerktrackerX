Scan QR Code

Scan order QR codes to track deliveries

How to Use

  • Tap the "Scan QR Code" button above
  • Position the QR code within the camera frame
  • Wait for the code to be recognized
  • You'll be automatically taken to the order tracking page

What You Can Scan

  • Order receipts with QR codes
  • Order tracking links
  • Shared order QR codes
